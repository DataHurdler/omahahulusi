# General Information

This folder contains all sheet music, in both staff notation and jianpu notation, for the Omaha Hulusi Ensemble. They are organized in sub-folders, with the numbers in the front of folder names indicate the order they were prepared:

- 1 Moonlight over the Lotus Pond
- 2 Longing for the Spring Breeze
- 3 Medley (Jolly Old Saint Nicholas/Amazing Grace/Oh! Susanna)
- 4 Boat on Tai Lake (in progress)
- 5 Purple Bamboo Medley (planned)
- 6 Dayung Sampan/甜蜜蜜 (planned)
- 99 Practice Tunes
